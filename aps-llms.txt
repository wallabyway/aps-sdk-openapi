# Autodesk Platform Services – OpenAPI Specs

## Core Authentication
- [Authentication API v2](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/authentication/authentication.yaml): OAuth 2.0 token & user profile endpoints
- [Secure Service Accounts (SSA)](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/authentication/ssa.yaml): Server‑to‑server auth flows
- [Secure Service Accounts Example for creating SA accounts, and generating 3LO access tokens in 3 languages (SSA)](https://developer.doc.autodesk.com/bPlouYTd/cloud-platform-ssa-docs-main-460369/ssa/v1/tutorials/getting-started-with-ssa/about-this-walkthrough.html)

## Data & Model
- [Data Management API](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/datamanagement/datamanagement.yaml): Hubs, projects, folders, versions & objects
- [Model Derivative API](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/modelderivative/modelderivative.yaml): Translate & extract metadata, convert designs
- [Object Storage Service (OSS) API](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/oss/oss.yaml): Low‑level buckets & objects CRUD

## Viewer SDK
- [Viewer SDK](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/modelderivative/viewer-v7.md): embed a 2D/3D Model viewer inside your web-pages. Based on a modified version of Three.js. viewer designs that have been processed by model derivative service (SVF1, SVF2 formats) otherwise loads glTF and OBJ files.

## Construction Cloud
- [ACC Account Admin API](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/construction/accountadmin/accountadmin.yaml): Manage BIM 360 / ACC accounts & projects
- [ACC Issues API](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/construction/issues/Issues.yaml): Create, assign & track field issues

## Platform Integration
- [Webhooks API](https://raw.githubusercontent.com/wallabyway/aps-sdk-openapi/main/webhooks/webhooks.yaml): Subscribe to event callbacks across APS services
